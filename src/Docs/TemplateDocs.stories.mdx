## Step 1:

Create a new file called `_templateName_.tsx` inside of the sub-folder `src/Components/Templates`.
This file is going to be the place where different React components are being put together in order
to form the new template.

<br />

## Step 2:

Start of editing `_templateName_.tsx` by adding the following two import-statements:

```jsx
import React from 'react';
import { Story } from '@storybook/react';
```
